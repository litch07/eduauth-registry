# EduAuth Registry

EduAuth Registry is a university certificate and document verification platform built with React, Laravel, and MySQL.

## Features

- Student certificate upload and management
- Institution and admin verification workflow
- Employer and third-party certificate verification
- MySQL-backed data storage
- Role-based access and API-driven frontend

## Tech Stack

- Frontend: React
- Backend: PHP Laravel
- Database: MySQL

## Project Structure

- `frontend/` - React UI
- `backend/` - Laravel API
- `database/` - SQL schema

## Setup

1. Import `database/eduauth_registry.sql` into MySQL.
2. Configure the backend `.env` file for your local database.
3. Start the backend and frontend apps from their respective folders.
