# Running the Tests

## Prerequisites
```bash
pip install selenium webdriver-manager
```

## Run all tests
```bash
python test_student_registration.py
python test_student_apply_university.py
python test_student_login_and_dashboard.py
python test_public_certificate_verification.py
python test_admin_user_management.py
```

## Notes
- Ensure frontend is running on `http://localhost:5173`
- Ensure backend is running on `http://localhost:8000`
- Ensure database is seeded with test accounts
- Chrome browser must be installed
